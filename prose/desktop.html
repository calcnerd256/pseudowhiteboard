Pseudowhiteboard

browser-based

client

desktop

Connect a laptop or desktop computer to a projector or large TV and open Pseudowhiteboard in your Web browser, so everyone can see the big picture.

The desktop version of the interface shows the URL where the mobile users can find the mobile version of the interface.
